# Amazon-Sales-Analytics
This project uses an Amazon sales dataset from Kaggle (128,975 rows and 24 columns) to predict sales amount using Linear Regression.
## What I Did
* Performed data cleaning and handled missing values using median imputation.
* Conducted basic Exploratory Data Analysis (EDA).
* Selected relevant features for training.
* Built and evaluated a Linear Regression model using Scikit-learn.
## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook.
## Key Learning
Gained hands-on experience with the complete Machine Learning workflow, including data preprocessing, EDA, model training, and prediction.
